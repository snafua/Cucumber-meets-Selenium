However, it's
        also possible to build BootsFaces from source. This way, you get our latest bug fixes and the newest components,
        and you can also contribute bug fixes or components yourself to the project.</P><P>The BootsFaces source code is hosted on a <A href="https://github.com/TheCoder4eu/BootsFaces-OSP/" target=_blank>GitHub repository</A>.
        Clone the repository using git (or download the zip file and unpack it to a folder). This folder contains two build files: a Maven <CODE>pom.xml</CODE> and an
        ANT <CODE>build.xml</CODE>. 
        </P><P>To build the jar, run <CODE>mvn clean install</CODE> on the command line. Most IDE also offer decent support for Maven. For instance, Eclipse users
        can import the project via "Import --&gt; Maven --&gt; Existing Maven project".</P><P> The ANT build file is more complete and more sophisticated than the Maven file. Among other things, it minifies the code, yielding slightly better performance. 
        It also extracts the CSS and Javascript files automatically from Bootstrap, and it allows you to build a customized version of BootsFaces containing only the
        code you really need. There's a detailed instruction in
